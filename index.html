О игре «Рабочий обзор»
Важно: баланс, копейки, рубли, доход в час и турнирные места — это игровые показатели. Это не настоящие деньги, их нельзя вывести, обменять, перевести или использовать как оплату.
Суть игры: игрок регистрируется по номеру телефона, выполняет игровые заказы, нажимает на рабочего, закрывает этапы задания, повышает уровень, копит игровой баланс и развивает пассивный доход.
Правила игры:
  • Каждый заказ состоит из этапов и дедлайна.
  • Нажатия тратят энергию и двигают прогресс заказа.
  • Если заказ выполнен — начисляется игровая награда.
  • Если заказ не успел выполниться — он обновляется без списания баланса и энергии.
  • Пассивный доход начисляется в игровых копейках по внутренней логике.
  • Турнир показывает рейтинг игроков по игровому балансу и уровню.
Политика конфиденциальности: имя, фамилия, номер телефона и прогресс сохраняются в localStorage браузера. Данные не отправляются на сервер, если в коде не указан отдельный API для онлайн-турнира. При очистке данных сайта профиль и прогресс могут быть удалены.
Онлайн-турнир: без подключённого API турнир работает локально в этом браузере. Для настоящего общего рейтинга между разными пользователями нужен сервер или база данных.
